# Misc / Uncategorized

Use this section when you have **something worth keeping** but you are **not sure** which main area it belongs in (Historical Record, Articles, Stats and Sheets, Video Library, Webpages, Image Library, and so on).

## How it works

1. Add a Markdown page here (or extend this overview with a short entry and links).
2. Open a pull request (see **[Submitting materials](../contributing/submitting-materials.md)**).
3. Maintainers can **move** the content later into the right section and fix `mkdocs.yml` so the sidebar stays tidy.

This is a **staging** area, not a junk drawer: still write a clear title, a sentence of context, and links or file paths so reviewers know what they are looking at.
