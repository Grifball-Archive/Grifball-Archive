# Statistics

Store **numbers people will want years later**: standings, Elo exports, sign-up sheets, etc.

## In this repository

- Prefer **CSV**, **PDF**, or small **XLSX** exports under `docs/assets/files/` with a Markdown page here explaining the columns and the event.
- For very large datasets, consider linking to a dedicated dataset host and keep a **description + checksum + date** in the archive.

## Google Sheets

If you keep a live sheet, link it **and** attach periodic exports to the repo so the record survives if the link changes.
