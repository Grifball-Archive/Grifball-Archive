# Using this archive

## Browse on the web

The published site offers **search**, a **sidebar outline**, and a **table of contents** per page — similar in spirit to [TSG Forge Wiki](https://wiki.thescriptersguild.com/) (GitBook).

## Download a full copy

| Method | What you get |
|--------|----------------|
| **Clone** | All Markdown, assets, and git history: `git clone https://github.com/Grifball-Archive/Grifball-Archive.git` |
| **ZIP** | Snapshot of `main`: [Download ZIP](https://github.com/Grifball-Archive/Grifball-Archive/archive/refs/heads/main.zip) |

The site content lives under **`docs/`** in that repository.

## What this is not

This archive does **not** host league operations (sign-ups, schedules, live brackets). It only preserves **information** others can read and cite later.
