# Articles

Long-form **essays, oral history, tournament write-ups**, and similar prose live in this section. Each piece should be **fully readable on the website** once merged—visitors should not need GitHub to understand the narrative.

## Adding a new article

1. Create `docs/articles/your-topic.md` in the repository.
2. Write a clear title (`#` heading) and sections. Link to statistics or VOD pages where relevant.
3. Add the file under **Articles** in **`mkdocs.yml`** so it appears in the sidebar.

!!! tip "Stable filenames"
    Prefer short, stable names (`2010-regionals-retrospective.md`) so links from other pages do not break when you tweak titles.
