# Webpages

Save **snapshots of the web**: linked articles, forum threads, event pages, and **screenshots** so the archive still has something to show if hosts go offline or URLs rot.

## What belongs here

- Short Markdown pages that **quote**, **summarize**, or **link** to a source, with a **capture date**.
- **Images** (PNG/JPEG) of pages or UIs under `docs/assets/images/` (or a subfolder you create), embedded from these pages.
- Optional **PDF exports** or HTML archives in `docs/assets/files/` when you have a stable export.

Keep files reasonably small; prefer linking plus a screenshot over duplicating huge HTML dumps unless there is a strong preservation reason.
