### Wiki (primary way to help)

1. Open [**Grifball-Archive/Grifball-Archive → Wiki**](https://github.com/Grifball-Archive/Grifball-Archive/wiki).
2. Sign in to GitHub.
3. Use **New page** or open a page → **Edit**.

Full guidance (articles, images, spreadsheets, video, sidebar): [**CONTRIBUTING.md**](https://github.com/Grifball-Archive/Grifball-Archive/blob/main/CONTRIBUTING.md) in the main repository.

### Improve navigation

- **Wiki sidebar:** add or edit **`_Sidebar.md`** in the wiki (easiest from a [wiki clone](https://github.com/Grifball-Archive/Grifball-Archive.wiki.git)) so readers can jump to major topics.
- **This Pages site:** new top-level pages can be added as `.md` files in the [main repo](https://github.com/Grifball-Archive/Grifball-Archive); the header nav is in **`_layouts/default.html`**.

### Main repository (optional)

Changes to the README, site theme, or `CONTRIBUTING.md` use normal **pull requests** on the [main repo](https://github.com/Grifball-Archive/Grifball-Archive) unless you have direct write access.
